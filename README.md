# Terrorscape

Звуковое сопровождение и фиксация статистики партий для настольной игры "Шепот за стеной" (Terrorscape).

## Progressive Web App (PWA)

Приложение настроено как PWA и может быть установлено на мобильные устройства для работы офлайн.

### Установка PWA

1. Откройте приложение в браузере (Chrome, Edge, Safari на iOS)
2. Перейдите в настройки
3. Нажмите кнопку "Установить приложение"
4. Или используйте меню браузера: "Добавить на главный экран" / "Install app"
